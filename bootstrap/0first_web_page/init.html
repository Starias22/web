Salut à tous

Boostrap est un framework front-end gratuit conçu pour réaliser les applications web plus rapidement et plus facilement

Il est constitué de design HTML, CSS de base pour les boutons, les formulaires, les images, les barres de navigation, etc

Il permet de réaliser des applications web qui s'adaptent à différentes machines, allant des petits appareils mobiles aux grands ordinateurs
